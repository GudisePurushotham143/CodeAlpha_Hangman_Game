# CodeAlpha Hangman Game
## Project Description

This is a simple Hangman Game developed using Python.
The player has to guess the correct word one letter at a time.
## Features

- Random word selection
- User letter guessing
- Maximum 6 incorrect attempts
- Win and Game Over messages
## Technologies Used

- Python
- Random Module
## How to Run

1. Open Terminal
2. Go to the Project Folder
3. Run:

python hangman.py
